# HCL Marketplace

## About

HCL Marketplace using nodejs, express and nextjs.

## Commands

### Dev Build

```node
npm install
npm start
```

### Test Build

```node
npm install
npm run build:test
npm run serve:test
```

### Production Build

```node
npm install
npm run build
npm run serve
npm run export
```

Dev Url: Open `http://localhost:3200/` in your browser to see your app running.
